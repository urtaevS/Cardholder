# Telegram Mini App - Карты лояльности

Мини-приложение для Telegram, позволяющее хранить, добавлять и использовать карты лояльности магазинов.

## Функционал

- ✅ Добавление карт лояльности (название + штрихкод)
- ✅ Сканирование штрихкода через камеру телефона
- ✅ Отображение штрихкода при нажатии на карту
- ✅ Удаление карты с двойным подтверждением
- ✅ Хранение данных в localStorage

## Технологии

- HTML5
- CSS3
- JavaScript
- Telegram WebApp API
- jsQR (для сканирования штрихкодов)
- JsBarcode (для отображения штрихкодов)

## Установка

1. Клонируйте репозиторий
2. Разместите файлы на веб-сервере (например, GitHub Pages)
3. Создайте Telegram-бота через @BotFather
4. Привяжите URL приложения к боту

## Использование

1. Откройте приложение через Telegram
2. Нажмите "Добавить карту"
3. Введите название магазина
4. Отсканируйте штрихкод или введите вручную
5. Сохраните карту
6. Нажмите на карту для просмотра штрихкода
